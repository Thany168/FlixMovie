# Movie App (Next.js + TypeScript + TMDB API)
A modern movie browsing web application built with **Next.js**, **TypeScript**, and **TMDB API**.  
This project is part of my learning journey to master **Next.js and API integration** with consistency and step-by-step development.
---
## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- TMDB API (The Movie Database)
- React Hooks
